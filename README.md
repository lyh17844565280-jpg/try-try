# It's a skill.
I distilled the books of Nietzsche and Schopenhauer and eventually obtained this skill.After directly downloading and decompressing this file, for users of codex, place the file on Windows: Under the C:\Users\< your username >\.codex\skills folder, you will obtain a Philosopy-Thinking file. This file is the skill, and on codex, the name of this skill is displayed as "哲学thinking".
